### Dependencies

```
- Frontend:Node v14.15.1
- Backend:Node v12.0.0

- Frontend:npm 6.14.8
- Backend:npm 6.9

- AWS CLI v2
- EB CLI

- A RDS database running Postgres.

- A S3 bucket for hosting uploaded pictures.

```
